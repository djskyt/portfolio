# 🌟 Popol — Frontend Portfolio

---

## 🔧 Tech Stack

- **React 19**, **TypeScript**, **Vite**
- **Tailwind CSS**, Lucide Icons

---

## 🚀 Getting Started

### 1️⃣ Clone & Install

```bash
git clone https://https://github.com/djskyt/portfolio
cd portfolio
npm install # or npm install
```

### 2️⃣ Run Dev Server

```bash
npm run dev
```

➡️ http://localhost:5173 자동 실행

### 3️⃣ Build

```bash
npm run build
```
